# TemanNakes

Aplikasi offline "TemanNakes" - Asisten obat esensial untuk tenaga kesehatan di Indonesia.

## Fitur
- **118 Database Obat**: Sesuai standar DOEN & Fornas.
- **Pencarian FTS5**: Super cepat dan typo-tolerant.
- **Kalkulator Dosis**: Presisi desimal untuk pediatrik.
- **Cek Interaksi**: Pindai potensi interaksi obat secara instan.
- **Favorit**: Simpan obat yang sering digunakan.
